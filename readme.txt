Add-On for WP Inventory - provides WP Inventory integration with Yoast SEO breadcrumbs and NavXT breadcrumbs

== Description ==

Causes the breadcrumbs to show the inventory item name when viewing an inventory item.

* Highly automatic.
* Integrates with Yoast SEO Breadcrumbs
* Integrates with NavXT Breadcrumbs

== Installation ==

1. Upload ‘wpinventory_breadcrumb_integration’ to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Under the “WP Inventory” menu, go to "Settings", then the "General" tab.
